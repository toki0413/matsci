# Crystallography Basics

## Lattices and Unit Cells

- A crystal is built by repeating a unit cell in three dimensions.
- 14 Bravais lattices grouped into 7 crystal systems.
- Lattice vectors `a, b, c` and angles `α, β, γ` define the unit-cell metric.

| Crystal system | Lattice condition | # Bravais | Laue group |
|---|---|---|---|
| Triclinic | a≠b≠c, α≠β≠γ | 1 | $\bar{1}$ |
| Monoclinic | a≠b≠c, α=γ=90°≠β | 2 | 2/m |
| Orthorhombic | a≠b≠c, α=β=γ=90° | 4 | mmm |
| Tetragonal | a=b≠c, α=β=γ=90° | 2 | 4/mmm |
| Trigonal | a=b=c, α=β=γ<120° | 1 | $\bar{3}m$ |
| Hexagonal | a=b≠c, α=β=90°, γ=120° | 1 | 6/mmm |
| Cubic | a=b=c, α=β=γ=90° | 3 | $m\bar{3}m$ |

## Miller Indices

- Planes: `(h k l)` represent reciprocal lattice intercepts.
- Directions: `[u v w]` represent direct-space vectors.
- In cubic systems, plane normals and directions with the same indices are parallel.
- Family of equivalent planes: `{h k l}`. Family of equivalent directions: `<u v w>`.

## Symmetry and Space Groups

- 230 space groups = point symmetry (32 crystallographic point groups) × translational symmetry.
- Translational symmetry operations: lattice translations, glide planes, screw axes.
- Space group symbol (Hermann-Mauguin): e.g. `Fm-3m` (#225), `P6_3/mmc` (#194).
- Two notations: Hermann-Mauguin (international, default) and Schoenflies (molecular).

## Wyckoff Positions (per space group)

A Wyckoff position is a set of symmetry-equivalent sites in a space group.
Each position has:
- **Multiplicity**: how many equivalent sites per unit cell
- **Letter**: `a, b, c, ...` (a = highest symmetry, lowest multiplicity)
- **Site symmetry**: subgroup of the point group that leaves the site invariant
- **Coordinates**: explicit list of (x, y, z) generated by symmetry

Example, space group Fm-3m (#225), face-centered cubic:

| Wyckoff | Mult | Site sym | Coordinates | Typical occupant |
|---|---|---|---|---|
| 4a | 4 | m-3m | (0,0,0) + fcc | Cu, Al |
| 4b | 4 | m-3m | (½,½,½) + fcc | — |
| 8c | 8 | -43m | (¼,¼,¼) + fcc | diamond, ZnS |
| 24d | 24 | -4m2 | (¼,¼,0) + fcc | — |
| 48h | 48 | m.. | (x,0,0) + fcc | — |

In rocksalt (NaCl, Fm-3m): Na at 4a, Cl at 4b.
In diamond (Fd-3m, #227): C at 8a.
In perovskite (Pm-3m, #221): A at 1a, B at 1b, O at 3c.

## Bilbao Crystallographic Server — query patterns

URL: https://www.cryst.ehu.es (academic free; cite Bilbao in publications).

Key tools and what they answer:

| Tool | Question it answers |
|---|---|
| WYCKPOS | Wyckoff positions of a space group (the table above) |
| SUBGROUPGRAPH | All subgroups of SG; minimal path between SG and subgroup |
| WYCKSPLIT | How a Wyckoff position splits under a subgroup (phase transition) |
| AMPLIMODES | Mode decomposition of a distorted structure (group-theoretical) |
| SYMMETRYMODES | Symmetry-adapted distortion modes |
| BANDS | Band representation / elementary band representations (topological) |
| IRREPS | Irreducible representations of a space group at any k-point |
| MAGNEXT | Magnetic space groups (1651 types, black-white + color) |
| PSEUDOLAT | Pseudosymmetry search (high-symmetry parent of distorted structure) |

## Subgroup chains and phase transitions

Every phase transition lowers symmetry: parent SG → subgroup SG. The chain
encodes the order parameter.

Example: BaTiO₃ cubic → tetragonal (ferroelectric)
- Parent: Pm-3m (#221)
- Subgroup: P4mm (#99)
- Index: 6 (i.e. 1/6 of sym ops retained)
- Order parameter: Ti displacement along [001]
- Tool: SUBGROUPGRAPH confirms minimal chain Pm-3m → P4mm; WYCKSPLIT shows
  1a (Ti, parent) → 1a (Ti, subgroup), no splitting of Ti site.

## Reciprocal Lattice

- Defined by vectors `a* = 2π (b × c) / V`, `b* = 2π (c × a) / V`, `c* = 2π (a × b) / V`
- Diffraction peaks occur when the scattering vector `Q = h a* + k b* + l c*`.
- For cubic lattice with constant a: `|Q| = 2π/d_hkl = (2π/a)·sqrt(h²+k²+l²)`.
- Brillouin zone = Wigner-Seitz cell of reciprocal lattice.

## Irreducible Brillouin Zone (IBZ)

- Symmetry-equivalent k-points reduce to a single representative inside IBZ.
- Number of IBZ k-points scales as 1/|point group order| (approx).
- DFT codes (VASP, QE) automatically reduce the mesh via symmetry.
- For hexagonal: IBZ is 1/12 of full BZ (6-fold rotation + mirror).
- Breaking symmetry (e.g. magnetic ordering, distortion) inflates IBZ → more k-points.

## Tools

- `spglib` (BSD-3, Python/C) — de facto standard for space-group detection from
  atomic positions. Returns space-group number, Wyckoff letters, symmetry ops.
- `pymatgen.symmetry` (MIT) — wraps spglib + adds Spacegroup, WyckoffSite classes.
- `moyo` (Rust+WASM) — newer, faster, used by CatGo.
- `Bilbao Crystallographic Server` — web queries, not a Python lib.
- `ISOTROPY` (Stokes, BYU) — suite for symmetry-mode analysis, complements Bilbao.

## Practical Notes

- Always report conventional vs. primitive cells when comparing structures.
- Check for fractional-coordinate wrapping and image conventions (pbc).
- Use Wyckoff positions to reduce the number of independent structural parameters.
- When distorting a structure, run WYCKSPLIT to know which atoms become independent.
- For magnetic structures, use magnetic space groups (BNS notation), not ordinary SG.
- Pseudosymmetry is common in real crystals — search for higher-symmetry parent.
