# Multi-Modal Visualization for Computational Materials

Effective visualization combines numerical plots, structural images, and text annotations so researchers can interpret complex simulation outputs at a glance.

## Plot Types by Data

- **Convergence plots**: energy / force / electronic-step vs. iteration; essential for DFT and MD validation.
- **Radial distribution functions (RDF)**: pair-distance histograms; compare liquid, amorphous, and crystalline phases.
- **Band structure + DOS**: energy vs. k-path and density of states; identify band gaps and orbital character.
- **Phonon dispersion**: frequency vs. q-point; detect imaginary modes and dynamic stability.
- **Pareto fronts**: trade-off plots for multi-objective optimization and active-learning campaigns.
- **Evolution dashboards**: bar charts, confidence histograms, and trend lines for skill/evolution reports.

## Multi-Modal Composition

A comprehensive report often merges:

1. **Static plots** generated by matplotlib/plotly (PNG/SVG/HTML).
2. **Structure renderings** from pymatgen/ASE/OVITO (PNG or interactive 3D viewers).
3. **Tables** of key quantities (lattice params, energies, gaps, elastic constants).
4. **Narrative text** explaining methods, results, and caveats.

## Best Practices

- Use consistent color scales and units across related figures.
- Label axes, include legends, and cite the computation method.
- Save vector formats (SVG/PDF) for publications; use HTML for interactive dashboards.
- Keep figure resolution ≥ 150 DPI for slides/reports and ≥ 300 DPI for publications.
- Embed image paths in Markdown/LaTeX reports so they render in the final document.

## Automation Tips

- Generate plots programmatically from JSON report files so they stay in sync with results.
- Use a single output directory per workflow (`workspace/<session>/figures/`).
- Name files descriptively: `band_structure.png`, `convergence_energy.png`, `pareto_front.svg`.
- Combine multiple subplots into one figure for side-by-side comparisons.

## Accessibility

- Provide alt-text or captions for every figure.
- Avoid color-only encoding; add line styles or markers.
- Ensure sufficient contrast for text and axis labels.

## Integration with Agent Workflows

A visualization skill can take a JSON report, decide the appropriate plot type, and emit figure paths. Those paths are then passed to a report tool to build the final Markdown/HTML document.
