Built for Real Research

Paper to Calculation

Huginn reads research papers, extracts simulation parameters, and sets up computational workflows automatically — turning hours of manual configuration into minutes.

  • Extracts DFT parameters from published papers
  • Auto-configures INCAR, POSCAR, KPOINTS
  • Validates input against known convergence criteria
$ huginn run --engine vasp --task relaxation
Setting up VASP relaxation...
Extracted params from DOI:10.1038/s41586
INCAR, POSCAR, KPOINTS ready
Job submitted to cluster gpu-01
$ huginn analyze --structure POSCAR --output report

Multi-Agent Collaboration

A team of specialized AI agents works together — one agent handles DFT, another manages molecular dynamics, a third synthesizes results into coherent reports.

  • Domain-specialized agent orchestration
  • Shared context across simulation steps
  • Automatic error recovery and retry logic
$ huginn agents list
dft-agent [active] VASP / QE workflows
md-agent [active] LAMMPS / GROMACS
analysis-agent [active] pymatgen / analysis
report-agent [idle] synthesis
$ huginn search --property bandgap --min 1.5eV

End-to-End Automation
